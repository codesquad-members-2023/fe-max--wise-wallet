1. 월요일 : 시멘틱 태그, CSS 레이아웃 학습
2. 화요일 : 메인페이지 UI 구현
3. 수요일 : UI 마무리, 내역 입력 기능 설계
4. 목요일 : 내역 입력 기능 구현, 월 이동 설계
5. 금요일 : 월 이동 구현
